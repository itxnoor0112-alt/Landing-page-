# NexTech Smart Watch Landing Page

## Project Overview
This project provides a professional, modern landing page for "NexTech" smart watches. It has been developed to showcase key product features, series models, and pricing in a clean, user-friendly interface.

## Key Features & Responsiveness
* **Fully Responsive:** The layout is optimized to look perfect on all screen sizes, including Mobile, Tablet, and Desktop using CSS Flexbox and Grid.
* **Modern UI/UX:** Features a sticky navigation header, intuitive product grid system, and interactive hover effects.
* **Functional Contact Form:** Includes a JavaScript-powered contact form that provides real-time success feedback upon submission.
* **Semantic HTML5:** Built with clean, accessible, and structured code.

## Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Responsive design techniques including Media Queries, Flexbox, and CSS Grid.
* **JavaScript (ES6):** DOM manipulation for interactive form handling.

## Project Structure
The project consists of two main files:
1. **index.html**  This contains the main landing page source code.
2. **README.md**  This document provides the project overview and setup instructions.

## How to Use
1. Clone or download the repository.
2. Open the **index.html** file in any modern web browser to view the project.
