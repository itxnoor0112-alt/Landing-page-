# NexTech Smart Watch Landing Page

## Project Overview

NexTech Smart Watch Landing Page is a responsive front-end web project developed using HTML5, CSS3, and JavaScript. The project showcases a modern smartwatch landing page with a clean user interface, responsive design, dark/light mode, and a contact form. The project follows a structured and organized approach to front-end web development.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Visual Studio Code
- Git
- GitHub

---

## Features

- Responsive design for Mobile, Tablet, and Desktop
- Semantic HTML5 structure
- Sticky Navigation Bar
- Hero Section
- Features Section
- Product Showcase
- Contact Form
- Contact Form Success Message
- Dark/Light Mode Toggle
- Smooth Scrolling
- Hover Effects
- Responsive Grid Layout
- External CSS and JavaScript Files
- Clean and Organized Project Structure

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open the index.html file in any modern web browser.
4. Explore the landing page and test all implemented features.
